# BackVora Improvements - 2026-02-20

## Summary
Enhanced the LinkBuilder app with CAPTCHA detection, bulk contact grabbing, contact summaries on the domains list, and improved dashboard stats.

## Changes Made

### 1. CAPTCHA Detection on Forms ✓

**Backend:**
- Added `has_captcha` (boolean) and `captcha_type` (string) columns to `ContactForm` model
- Updated `scraper.py` `_detect_forms()` method to:
  - Detect reCAPTCHA (v2 and v3) by checking for `g-recaptcha`, `grecaptcha`, `recaptcha` patterns
  - Detect hCaptcha by checking for `h-captcha`, `hcaptcha` patterns
  - Store CAPTCHA type: `recaptcha_v2`, `recaptcha_v3`, `hcaptcha`, or `none`
- Updated contacts router to save `has_captcha` and `captcha_type` when saving forms

**Frontend:**
- Updated `DomainDetailPage.tsx` to show:
  - "⚠️ CAPTCHA (type)" warning badge next to forms with CAPTCHA
  - Disabled auto-submit button for forms with CAPTCHA
  - Added "Open in Browser" link for CAPTCHA forms instead of Submit button

**Database:**
- Migration executed: Added `has_captcha` and `captcha_type` columns to `contact_forms` table

### 2. Bulk Grab Contacts ✓

**Backend:**
- Added `POST /api/v1/contacts/grab-bulk` endpoint in `contacts.py`
  - Accepts optional `domain_ids` parameter
  - If no IDs provided, grabs all domains without any contacts
  - Rate limiting: 2 second delay between domains
  - Returns progress stats: processed count, contacts added, forms detected

**Frontend:**
- Added `bulkGrabContacts()` to `api.ts`
- Added "Grab All Missing" button on `DomainsPage.tsx`
  - Shows spinner animation while processing
  - Displays progress notification
  - Refreshes domain list after completion

### 3. Contacts Summary on Domains List ✓

**Backend:**
- Updated `domains.py` list endpoint to include contact stats via batch queries:
  - `contacts_count`: Number of contacts per domain
  - `has_primary_contact`: Whether domain has a primary contact
  - `has_email`: Whether domain has email contacts
  - `has_form`: Whether domain has detected forms
  - `has_captcha`: Whether domain has forms with CAPTCHA

**Frontend:**
- Updated `types.ts` to include contact stat fields in `Domain` interface
- Added "Contacts" column to `DomainsPage.tsx` table showing:
  - Contact count (green if has primary, gray otherwise)
  - 📧 icon if has email
  - 📝 icon if has form
  - ⚠️ icon if has CAPTCHA

### 4. Dashboard Stats Enhancement ✓

**Frontend:**
- Updated `DashboardPage.tsx` with new "Contacts Overview" section:
  - **Domains with Contacts** (green)
  - **Domains without Contacts** (orange)
  - **Forms Detected** (blue)
  - **Forms with CAPTCHA** (yellow)
- Stats calculated from domains list data

### 5. General Polish ✓

**All Changes Include:**
- Loading spinners for async operations (bulk grab, form submission)
- Toast notifications for all actions
- Error handling with try/catch blocks
- Graceful fallbacks for missing data
- Consistent dark theme (bg-gray-800, pink-500/600 accents)
